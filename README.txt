# ColorSorter
ColorSorter - программа для сортировки цветных объектов

Описание:
Консольное приложение для сортировки объектов по цвету (Красный, Синий, Зеленый)
с возможностью задания пользовательского порядка сортировки.

Структура проекта:
└───ColorSorter - основное приложение
    ├───Models
    │       ColorObject.cs          # Перечисление цветов (Red, Blue, Green)
    │
    ├───Services
    │       ColorSorter.cs          # Логика сортировки цветов
    │
    ├───Tests
    │       ColorSort.Tests         # Тесты
    │           ColorSortTest.cs    # Тесты сортировки (3 тест-кейса)
    │
    ├───Program.cs                  # Главный исполняемый файл
    ├───ColorSort.csproj            # Файл проекта

Как использовать:
1. Запустите программу
2. Введите цвета для сортировки (например: КСЗК)
3. Введите порядок сортировки (например: З>К>С)
4. Получите результат (например: ЗККС)

Пример работы:
Вход: КСЗК (Красный, Синий, Зеленый, Красный)
Порядок: З>К>С
Результат: ЗККС
